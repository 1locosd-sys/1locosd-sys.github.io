About

Curated Web Directory is an independently maintained directory of useful external resources.

Our small editorial team reviews pages across the web and groups the most useful ones into themed selections, adding a short, plain-language description to each so readers can decide quickly whether a resource is worth their time.

We are not affiliated with the sites we link to, and all outbound links point to their respective owners. If you believe a listing should be updated or removed, the links in the footer will help you navigate the site.
